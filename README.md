# ProteinOS

ProteinOS is an AI-powered decision engine that helps busy professionals achieve their daily protein goals with the fewest possible decisions.

---

## Vision

Instead of tracking calories, ProteinOS answers one simple question:

> "What's the best thing I can eat right now to complete my protein goal?"

---

## Tech Stack

| Layer | Technology |
|--------|------------|
| Frontend | Next.js |
| Backend | FastAPI |
| Database | PostgreSQL |
| Database GUI | pgAdmin |
| Containerization | Docker Desktop |
| Version Control | Git & GitHub |
